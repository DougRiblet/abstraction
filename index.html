Image Search Abstraction Layer

How to Use This Service

Send a search phrase as parameter to https://fathomless-reef-60319.herokuapp.com/search/.

This service will return ten results from Google Image Search in JSON format. Each result item includes image URL, thumbnail URL, webpage source URL and short text description.

By default, this service returns the first ten results. You may specify a different "page" of results by adding ?offset=[number] to your query. For example, /search/ducklings?offset=2 will give you results 11 through 20 for search term "ducklings".

Do not use question marks in your search term. This service interprets "?" as the end of your search term.

Recent Searches

You can get the ten most recent search terms, also in JSON format, at https://fathomless-reef-60319.herokuapp.com/recent/. (Link)
